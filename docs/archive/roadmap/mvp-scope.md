# MVP Scope

> Archived document.
> This was an earlier scope snapshot. It is kept for history, not as the current source of truth.
> For current scope boundaries, use `../../product/canonical-product-blueprint-zh.md` and `../../../AGENTS.md`.

## Must Have

- Coach Inbox as the main product surface
- guided interview first-run flow
- persistent material inbox
- structured profile state with field statuses
- profile patch proposal and confirmation flow
- Monthly Brief generation
- visible explanation of how new materials affect profile and recommendations

## Should Wait

- deeper school data enrichment
- richer opportunity discovery
- more advanced profile simulator
- guardian collaboration flows beyond basic household support
- analytics and observability platforms beyond basic app logging

## Explicitly Out Of Scope

- heavy structured intake wizard
- counselor-facing workbench
- full live school crawling
- complex OCR and universal transcript parsing
- essay ghostwriting
- mock interview system
- full chancing engine as the primary user value
