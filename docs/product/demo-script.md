# Demo Script

## Objective

Show that AdmitGenie is an AI-native admissions coach, not a static form tool or generic chat bot.

## Demo Length

- Target: 3 minutes
- Stretch: 5 minutes

## Story Arc

### Step 1: Start With A Real Conversation

Show the first-run coach inbox.

Narration:

> Instead of making families fill out a giant intake form, AdmitGenie starts like a coach. It asks for just enough context to start helping.

What to show:

- welcome message
- first user response
- one or two smart follow-up questions

### Step 2: Show Early Value

After a few turns, show:

- an initial understanding summary
- one likely hook direction
- one top priority for the week

Narration:

> The system already has enough context to say what matters now, even before the profile is complete.

### Step 3: Add New Material

User uploads or pastes a new material item.

Best demo candidate:

- SAT score update
- transcript note
- extracurricular update

Narration:

> Families can add new materials anytime. The system doesn't just store them; it updates understanding.

### Step 4: Show Structured Update

Show:

- extracted facts
- proposed profile patch
- human-readable summary

Narration:

> New information becomes structured profile state behind the scenes, with visible explanations instead of silent changes.

### Step 5: Show New Brief

Show updated Weekly Brief:

- what changed
- what matters this week
- top actions

Narration:

> The value isn't the upload itself. The value is that the advice changes when the student's profile changes.

## Demo Must Prove

- no giant intake form is required
- structure still exists behind the scenes
- new materials can be added anytime
- profile state is traceable and explainable
- recommendations update based on evidence

## Demo Must Not Depend On

- live crawling
- production OCR
- full school monitoring infrastructure
- counselor workflows
- long setup or account management flows

## Preferred Inputs For Demo

- grade level
- rough goal
- biggest concern
- one test score update or activity update

## Close

Suggested closing line:

> AdmitGenie is not just answering questions. It's maintaining a living admissions profile and turning that profile into weekly action.
