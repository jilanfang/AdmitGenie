# ADR 0001: MVP Baseline

## Status

Accepted

## Decision

The AdmitGenie MVP is defined as:

- an AI-native admissions coach
- for North America-focused 11th-grade families
- using a Coach Inbox
- centered on a Weekly Brief

## Why

This is the narrowest version of the product that still expresses the core idea:

- persistent coaching
- evolving profile state
- actionable weekly guidance

## Consequences

- counselor-first workflows are deferred
- school district workflows are deferred
- the product is optimized for a family-facing experience first
